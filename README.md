# 👾 FIFA Survivor

**FIFA Survivor** is a fast-paced, web-based arcade survival game. You play as a stressed organizer dodging an endless, scaling swarm of "bugs" and "syntax errors." Survive as long as you can, rack up the highest score, and keep the system from crashing!

Play it live here: https://mdfahimhassan.github.io/BUCC-Survivor/

## 🎮 Features
* **Auto-Fire Combat:** Focus purely on movement and dodging; your character automatically targets and blasts the nearest bug with blue energy balls.
* **Infinite Scaling:** The bugs don't stop coming. Test your reflexes in a highly chaotic, "Vampire Survivors" style loop.
* **Mobile-Ready:** Fully playable on a desktop browser (Keyboard) or on a smartphone (Touch & Drag). 
* **Juicy Arcade Feel:** Features particle explosions, screen shakes, and smooth movement.

## 🕹️ Controls
* **Desktop:** `W A S D` or `Arrow Keys` to move.
* **Mobile:** Tap and drag anywhere on the screen to move the character. 

## 🛠️ Tech Stack
This game was built using vanilla web technologies and requires no backend or build steps.
* **HTML5 / CSS3**
* **JavaScript** * **[Kaboom.js](https://kaboomjs.com/)** (v3000) - A fast and fun JavaScript game library.
